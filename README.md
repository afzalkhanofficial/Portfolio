## 🎯 Overview

This is a **modern, premium, dark-themed portfolio website** designed to make a striking first impression. Built entirely from scratch without any framework overhead — just pure HTML5, CSS3, and vanilla JavaScript — it features:

- 🌑 **Ultra-dark glassmorphism design** with noise textures and animated gradients
- ✨ **Micro-animations** — cursor glow, floating particles, scroll reveals, typewriter effects
- 📊 **Animated skill rings** with category filtering
- 🧪 **6 cutting-edge project showcases** spanning MERN, AI/ML, NLP & Design
- 💼 **3 internship experiences** on a color-coded timeline
- ⭐ **Supervisor testimonials** with star ratings
- 📬 **Contact form** with loading states and success feedback

## 📱 Responsive Design

The website is fully responsive with **5 breakpoints** and optimizations for:

- ✅ Desktop (1200px+)
- ✅ Laptops & Tablets landscape (992px – 1200px)
- ✅ Tablets portrait (768px – 992px)
- ✅ Mobile phones (576px – 768px)
- ✅ Small phones (375px – 576px)
- ✅ Extra small screens (< 375px)

Key responsive features:
- Collapsible hamburger navbar
- Stacked grid layouts on mobile
- Adjusted typography scales
- Touch-friendly interactive elements
- Optimized spacing and padding

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools or dependencies required!

### Run Locally

```bash
# Clone the repository
git clone https://github.com/your-username/portfolio.git

# Navigate to the project
cd portfolio

# Open in browser
# Option 1: Simply double-click index.html
# Option 2: Use VS Code Live Server extension
# Option 3: Use any local server
npx serve .
```

### Deploy

This is a **static website** — deploy it anywhere:

| Platform | Command / Steps |
|----------|----------------|
| **GitHub Pages** | Push to `main` → Settings → Pages → Deploy from branch |
| **Netlify** | Drag & drop the folder or connect your repo |
| **Vercel** | Import repo → Auto-deploy |
| **Firebase** | `firebase init hosting` → `firebase deploy` |
